## Work in progress

### [Authorship Identification ML Service](https://github.com/kirill-rubashevskiy/mlds23-authorship-identification)

#### Project Overview

As part of my master’s degree in Artificial Intelligence, I developed an ML service designed to identify 
the authorship of texts among ten prominent Russian 19th century writers. 

#### Main Goals

- **Build an End-to-End ML Pipeline**: From data collection from open sources to monitoring the performance of a 
deployed service.
- **Apply of ML Techniques in NLP**: Employ cutting-edge machine ML 
algorithms specifically tailored for natural language processing (NLP).
- **Get Hands-On MLOps and DevOps Experience**: Gain practical experience in MLOps and DevOps principles, including 
cloud storage, microservice architecture, containerization, deployment, testing, and performance monitoring.

#### Project Outcomes

- **Best Model Architecture and Performance**: Developed a logistic regression model over text statistics and 
Bag-of-Words embeddings that outperformed other ML algorithms with weighted F1 score 0.85 on test data.
- **Service Architecture and Deployment**: Build and containerized a service that includes a 
[FastAPI application](https://mlds23-authorship-identification-app.onrender.com/) to make 
single and batch predictions and a user-friendly [Telegram bot](https://t.me/mlds23_ai_bot); successfully deployed 
the service on Render.com.
